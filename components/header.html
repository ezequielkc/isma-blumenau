Exemplo do Componente Header

Este é um exemplo de como usar o componente de cabeçalho modular.

Como usar:

1. Método Manual (HTML estático)

Copie o código HTML do cabeçalho acima e cole em suas páginas.

2. Método Programático (JavaScript)


// Criar uma nova instância do template
const headerTemplate = new HeaderTemplate({
  logoPath: '/assets/',
  logoUff: 'logo-uff.jpg',
  logoIsma: 'logo-isma-site.png'
});

// Gerar o HTML
const headerHTML = headerTemplate.generate();

// Inserir no DOM
document.body.insertAdjacentHTML('afterbegin', headerHTML);

// Inicializar funcionalidade
const headerComponent = new HeaderComponent();

3. CSS e JavaScript necessários

Inclua os arquivos:

  • /css/header.css - Estilos do componente
  • /js/header.js - Funcionalidade JavaScript
  • Font Awesome para os ícones

Características do Componente:

  • ✅ Totalmente responsivo
  • ✅ Acessibilidade completa (ARIA, foco, teclado)
  • ✅ Menu mobile com overlay
  • ✅ CSS com variáveis customizáveis
  • ✅ JavaScript modular e reutilizável
  • ✅ Semântica HTML apropriada
  • ✅ Performance otimizada
